INSTALLING
To install the application on your device, you must have xCode version 26.1 or higher and iOS 26.1 or higher. Copy the repository to your Mac using the following command
git clone …

Open the project in Xcode. Then tap on the project and dive into the General settings menu in the App Icons and Launch Screen section. In the App Icon option, enter “AppIcon”. In the Supported Destination option, leave iOS.
Connect your device to your Mac via a USB cable, then select it as the target in the top menu.
Press ⌘ cmd R to deploy the application to the device.
First, open the iPhone system settings and activate Developer Mode at the very bottom. Go to General => VPN & Device Management and specify trust for Developer App

HOW TO USE
Swipe up and down to scroll through tasks. Press and hold on a task to enter edit mode. Add new tasks by pressing the ‘+’ button.
Swipe left to move a task to completed. Tap on the gallery and view the current and completed tabs. You can restore or completely delete tasks by swiping right and left, respectively.
Text format is supported, as well as text for copying — just tap on the text and it will be copied automatically, which is convenient for links, for example.

In a future version, it may be possible to add illustrations and select the language.
